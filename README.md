# AarogyaCare Enterprise ERP

AarogyaCare is an advanced, premium, multi-tenant ready hospital ERP, Clinical Decision Support System (CDSS), Business Intelligence Analytics dashboard, and Visual CMS platform.

## Key Modules
1. **Clinical Decision Support System (CDSS):** Symptom checker, drug contraindications analyzer, and Gemini-based medical assistant.
2. **Business Intelligence (BI) Dashboard:** Realtime Chart.js revenue dashboards, seasonal forecasting models, and comprehensive security audit sessions checker.
3. **Visual Page Builder (CMS):** Dynamic visual block-based layouts editor, newsrooms, events listings, job tracking, and SEO manager.
4. **General Hospital ERP:** Billing, inventory, pharmacy, HRMS, and ICU/OT registries.
