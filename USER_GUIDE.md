# User Operating Manual

This guide describes how to use AarogyaCare dashboards.

## Core Dashboards
1. **Doctor Portal:** Access patient EMR documents, write prescriptions, and use the Clinical AI Co-Pilot assistant tool.
2. **Patient Portal:** Book OPD visits, check diagnostic laboratory reports, and interact with the public triage symptom checker.
3. **Billing Console:** Review outstanding insurance claims and generate invoices.
