# Administrator Operating Manual

This guide describes how to configure the AarogyaCare platform.

## Configuration Items
1. **CMS Page Building:** Access `/cms/pages` to modify sections, heroes, and blocks of pages.
2. **SEO Redirects:** Access `/cms/seo` to set up 301 paths.
3. **KPI Management:** Edit target thresholds inside the analytics interface to monitor achievement rates.
4. **Security Audits:** Check the logs at `/analytics/audit` to track user login/logout details and database mutations.
